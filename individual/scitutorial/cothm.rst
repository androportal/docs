


cothm
=====

matrix hyperbolic cotangent



Calling Sequence
~~~~~~~~~~~~~~~~


::

    [t]=cothm(x)




Arguments
~~~~~~~~~

:x square matrix (real or complex).
: :t square matrix (real or complex).
:



Description
~~~~~~~~~~~

`cothm(x)` is the matrix hyperbolic cotangent of the square matrix x.



Examples
~~~~~~~~


::

    A=[1,2;3,4];
    cothm(A)
