


strrep (Matlab function)
========================

String search and replace



Matlab/Scilab equivalent
~~~~~~~~~~~~~~~~~~~~~~~~
**Matlab** **Scilab**

::

    strrep



::

    `strsubst`_




Particular cases
~~~~~~~~~~~~~~~~

Note that Matlab **strrep** can be use with not string inputs, what
Scilab **strsubst** can not. In this case use **mtlb_strrep** instead.
