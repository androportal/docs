


triu
====

upper triangle



Description
~~~~~~~~~~~

Upper triangle. See tril.



Examples
~~~~~~~~


::

    s=`poly`_(0,'s');
    triu([s,s;s,1])
    triu([1/s,1/s;1/s,1])
