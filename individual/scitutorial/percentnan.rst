


%nan
====

not-a-number



Calling Sequence
~~~~~~~~~~~~~~~~


::

    %nan




Description
~~~~~~~~~~~

`%nan` returns the IEEE double representation for Not-a-Number (NaN).



Example
~~~~~~~


::

    A = %nan
          B = %nan
          A == B
          `isnan`_(A)
          `isnan`_(B)
