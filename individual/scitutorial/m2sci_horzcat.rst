


horzcat (Matlab function)
=========================

Horizontal concatenation



Matlab/Scilab equivalent
~~~~~~~~~~~~~~~~~~~~~~~~
**Matlab** **Scilab**

::

    horzcat



::

    No equivalent




Particular cases
~~~~~~~~~~~~~~~~

Scilab equivalent for Matlab **horzcat(A1,A2,...,An)** is
**[A1,A2,...,An]** if all **Ai** are not character strings, else,
Scilab equivalent is **A1+A2+...+An**.
