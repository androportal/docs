====
"cothm"
====

Scilab Function Last update : April 1993
**cothm** - matrix hyperbolic cotangent



Calling Sequence
~~~~~~~~~~~~~~~~

[t]=cothm(x)




Description
~~~~~~~~~~~

**cothm(x)** is the matrix hyperbolic cotangent of the square matrix
x.



Examples
~~~~~~~~


::

    
    
    A=[1,2;3,4];
    cothm(A)
