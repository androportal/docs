====
"not"
====

Scilab Function Last update : April 1993
**not** - (~) logical not



Calling Sequence
~~~~~~~~~~~~~~~~

~A




Description
~~~~~~~~~~~

**~A** gives the element-wise negation of the elements of the boolean
matrix **A** .



Examples
~~~~~~~~


::

    
    
    ~[%t %t %f]
