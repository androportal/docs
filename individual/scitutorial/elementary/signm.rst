"signm"
====

Scilab Function Last update : April 1993
**signm** - matrix sign function



Description
~~~~~~~~~~~

For square and Hermitian matrices **X=signm(A)** is matrix sign
function.



Examples
~~~~~~~~


::

    
    
    A=rand(4,4);B=A+A';X=signm(B);spec(B),spec(X)
