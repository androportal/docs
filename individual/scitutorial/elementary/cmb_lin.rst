====
"cmb_lin"
====

Scilab Function Last update : April 1993
**cmb_lin** - symbolic linear combination



Calling Sequence
~~~~~~~~~~~~~~~~

[x]=cmb_lin(alfa,x,beta,y)




Description
~~~~~~~~~~~

Evaluates **alfa*x-beta*y** . ** alfa, beta, x, y** are character
strings. (low-level routine)
