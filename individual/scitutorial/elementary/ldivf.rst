====
"ldivf"
====

Scilab Function Last update : April 1993
**ldivf** - left symbolic division



Calling Sequence
~~~~~~~~~~~~~~~~

ldivf('d','c')




Description
~~~~~~~~~~~

returns the string **'c\d'** Trivial simplifications such as **'1\c' =
'c'** are performed.



Examples
~~~~~~~~


::

    
    
    ldivf('1','1')
    ldivf('a','0')
    ldivf('a','x')
    ldivf('2','4')
