"tanh"
====

Scilab Function Last update : April 1993
**tanh** - hyperbolic tangent



Calling Sequence
~~~~~~~~~~~~~~~~

t=tanh(x)




Description
~~~~~~~~~~~

the elements of **t** are the hyperbolic tangents of the elements of
**x**



Examples
~~~~~~~~


::

    
    
    x=[1,%i,-1,-%i]
    tanh(x)
    sinh(x)./cosh(x)
