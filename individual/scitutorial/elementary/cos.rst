Scilab Function Last update : April 1993
**cos** - cosine function



Calling Sequence
~~~~~~~~~~~~~~~~

[y]=cos(x)




Parameters
~~~~~~~~~~


+ **x** : real or complex vector/matrix




Description
~~~~~~~~~~~

For a vector or a matrix, **cos(x)** is the cosine of its elements .
For matrix cosine use **cosm(X)** function.



Examples
~~~~~~~~


::

    
    
    x=[0,1,%i]
    acos(cos(x))
