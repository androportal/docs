====
"eval"
====

Scilab Function Last update : April 1993
**eval** - evaluation of a matrix of strings



Calling Sequence
~~~~~~~~~~~~~~~~

[H]= eval(Z)




Description
~~~~~~~~~~~

returns the evaluation of the matrix of character strings **Z** .



Examples
~~~~~~~~


::

    
    
    a=1; b=2; Z=['a','sin(b)'] ; eval(Z)  //returns the matrix [1,0.909];
