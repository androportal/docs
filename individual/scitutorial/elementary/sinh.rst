Scilab Function Last update : April 1993
**sinh** - hyperbolic sine



Calling Sequence
~~~~~~~~~~~~~~~~

[t]=sinh(x)




Parameters
~~~~~~~~~~


+ **x,t** : real or complex vectors/matrices




Description
~~~~~~~~~~~

the elements of vector **t** are the hyperbolic sine of elements of
vector **x** .



Examples
~~~~~~~~


::

    
    
    asinh(sinh([0,1,%i]))
