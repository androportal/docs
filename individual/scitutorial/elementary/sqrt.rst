====
"sqrt"
====

Scilab Function Last update : April 1993
**sqrt** - square root



Calling Sequence
~~~~~~~~~~~~~~~~

y=sqrt(x)




Parameters
~~~~~~~~~~


+ **x** : real or complex scalar or vector




Description
~~~~~~~~~~~

**sqrt(x)** is the vector of the square root of the **x** elements.
Result is complex if **x** is negative.



Examples
~~~~~~~~


::

    
    
    sqrt([2,4])
    sqrt(-1)
