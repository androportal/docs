**log10** - logarithm



Calling Sequence
~~~~~~~~~~~~~~~~

y=log10(x)




Parameters
~~~~~~~~~~


+ **x** : vector or matrix




Description
~~~~~~~~~~~

decimal logarithm.If **x** is a vector
**log10(x)=[log10(x1),...,log10(xn)]** .



Examples
~~~~~~~~


::

    
    
    10.^log10([1,%i,-1,-%i])
