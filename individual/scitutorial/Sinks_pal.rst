


Sinks_pal
=========

Sinks palette



Block Screenshot
~~~~~~~~~~~~~~~~





Module
~~~~~~


+ `xcos`_




Description
~~~~~~~~~~~

In Sinks palette, you can find a variety of blocks used to display
(Scope) and write data during simulation and also some output ports
used in superblocks.

The blocks of that palette does not have regular output ports.



Blocks
~~~~~~
