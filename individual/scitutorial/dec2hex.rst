


dec2hex
=======

convert from decimal to hexadecimal



Calling Sequence
~~~~~~~~~~~~~~~~


::

    h=dec2hex(d)




Arguments
~~~~~~~~~

:d matrix of doubles, positive integer values
: :h matrix of strings
:



Description
~~~~~~~~~~~

`dec2hex(x)` returns the hexadecimal representation of a matrix of
integers.



Examples
~~~~~~~~


::

    dec2hex([2748 10;11 3])
