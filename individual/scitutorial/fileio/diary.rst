====
"diary"
====

Scilab Function Last update : April 1993
**diary** - diary of session



Calling Sequence
~~~~~~~~~~~~~~~~

diary('file-name')




Description
~~~~~~~~~~~

**diary** creates a file which contains a copy of the current Scilab
session. **diary(0)** interrupts the diary.
