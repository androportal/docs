


detr
====

polynomial determinant



Calling Sequence
~~~~~~~~~~~~~~~~


::

    d=detr(h)




Arguments
~~~~~~~~~

:h polynomial or rational square matrix
:



Description
~~~~~~~~~~~

`d=detr(h)` returns the determinant `d` of the polynomial or rational
function matrix `h`. Based on Leverrier's algorithm.
