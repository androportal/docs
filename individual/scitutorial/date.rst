


date
====

Current date as date string



Calling Sequence
~~~~~~~~~~~~~~~~


::

    dt=date()




Arguments
~~~~~~~~~

:dt a string
:



Description
~~~~~~~~~~~

dt=date() returns a string containing the date in dd-mmm-yyyy format.



Examples
~~~~~~~~


::

    date()
