


demo_message
============

display a message



Calling Sequence
~~~~~~~~~~~~~~~~


::

    demo_message(fil)




Description
~~~~~~~~~~~

The function demo_message displays the text message within the file
'fil' given in argument.



Examples
~~~~~~~~


::

    demo_message('SCI/demos/control/pid/pid_3.sce');
