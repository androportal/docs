


uint8 (Matlab function)
=======================

Convert to 8-bit unsigned integer



Matlab/Scilab equivalent
~~~~~~~~~~~~~~~~~~~~~~~~
**Matlab** **Scilab**

::

    `uint8`_



::

    `uint8`_




Particular cases
~~~~~~~~~~~~~~~~

For infinite values, Scilab and Matlab **uint8** return different
values.



Examples
~~~~~~~~
**Matlab** **Scilab**

::

    A = `uint8`_(inf)
    A = 255



::

    A = `uint8`_(%inf)
    A = 0
