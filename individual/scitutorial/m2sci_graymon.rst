


graymon (Matlab function)
=========================

Set graphics defaults for gray-scale monitors



Matlab/Scilab equivalent
~~~~~~~~~~~~~~~~~~~~~~~~
**Matlab** **Scilab**

::

    graymon



::

    No equivalent




Particular cases
~~~~~~~~~~~~~~~~

This Matlab function can be replaced in Scilab by:
**set(gdf(),"color_map",[0.75,0.5,0.25]'*ones(1,3))**.
