


consolebox
==========

show or hide console box (Windows)



Calling Sequence
~~~~~~~~~~~~~~~~


::

    state1=console(state2)
    state1=console()




Arguments
~~~~~~~~~

:state1 returns console's state 'on' or 'off'
: :state2 'on' or 'off' set console's state 'toggle' to switch between
  console's state
:



Description
~~~~~~~~~~~

show or hide console box (Windows).



Examples
~~~~~~~~


::

    consolebox('off')
    state=consolebox()
    consolebox()
    consolebox('toggle')
    consolebox('toggle')
