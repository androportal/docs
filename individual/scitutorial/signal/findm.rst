====

Scilab Function Last update : April 1993
**findm** - for elliptic filter design



Calling Sequence
~~~~~~~~~~~~~~~~

[m]=findm(chi)




Description
~~~~~~~~~~~

Search for **m** such that **chi = %k(1-m)/%k(m)** (For use with
**find_freq** ).
