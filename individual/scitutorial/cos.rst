


cos
===

cosine function



Calling Sequence
~~~~~~~~~~~~~~~~


::

    [y]=cos(x)




Arguments
~~~~~~~~~

:x real or complex vector/matrix
:



Description
~~~~~~~~~~~

For a vector or a matrix, `cos(x)` is the cosine of its elements . For
matrix cosine use `cosm(X)` function.



Sample
~~~~~~



Examples
~~~~~~~~


::

    x=[0,1,%i]
    `acos`_(cos(x))




See Also
~~~~~~~~


+ `cosm`_ matrix cosine function
+ `acos`_ element wise cosine inverse (radians)
