


intc
====

Cauchy integral



Calling Sequence
~~~~~~~~~~~~~~~~


::

    [y]=intc(a,b,f)




Arguments
~~~~~~~~~

:a, b two complex numbers
: :f "external" function
:



Description
~~~~~~~~~~~

If `f` is a complex-valued function, `intc(a,b,f)` computes the
integral from `a` to `b` of `f(z)dz` along the straight line `a b` of
the complex plane.
