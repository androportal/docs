


dec2oct
=======

convert from decimal to octal



Calling Sequence
~~~~~~~~~~~~~~~~


::

    o=dec2oct(d)




Arguments
~~~~~~~~~

:d matrix of doubles, positive integer values
: :o matrix of strings
:



Description
~~~~~~~~~~~

`dec2oct(x)` returns the octal representation of a matrix of integers.



Examples
~~~~~~~~


::

    dec2oct([2748 10;11 3])
