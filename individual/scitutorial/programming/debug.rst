====
"debug"
====

Scilab Function Last update : April 1993
**debug** - debugging level



Calling Sequence
~~~~~~~~~~~~~~~~

debug(level-int)
level-int=debug()




Parameters
~~~~~~~~~~


+ **level-int** : integer (0 to 4)




Description
~~~~~~~~~~~

For the values 0,1,2,3,4 of **level-int** , **debug** defines various
levels of debugging. (For Scilab experts only).
