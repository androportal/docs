====
"date"
====

Scilab Function Last update : Jun 1999
**date** - Current date as date string



Calling Sequence
~~~~~~~~~~~~~~~~

dt=date()




Parameters
~~~~~~~~~~


+ **dt** : a string




Description
~~~~~~~~~~~

dt=date() returns a string containing the date in dd-mmm-yyyy format.



Examples
~~~~~~~~


::

    
    
    date()
