====
"null"
====

Scilab Function Last update : April 1993
**null** - delete an element in a list



Calling Sequence
~~~~~~~~~~~~~~~~

l(i)=null()




Description
~~~~~~~~~~~

Deletion of objects inside a list



Examples
~~~~~~~~


::

    
    
    l=list(1,2,3);
    l(2)=null() // get list(1,3)
