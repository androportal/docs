====
"break"
====

Scilab Function Last update : April 1993
**break** - keyword to interrupt loops



Description
~~~~~~~~~~~

Inside a **for** or **while** loop, the command **break** forces the
end of the loop.



Examples
~~~~~~~~


::

    
    
    k=0; while 1==1, k=k+1; if k > 100 then  break,end; end
