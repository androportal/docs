


real
====

real part



Calling Sequence
~~~~~~~~~~~~~~~~


::

    [y]=real(x)




Arguments
~~~~~~~~~

:x real or complex vector or matrix
: :y real matrix
:



Description
~~~~~~~~~~~

`real(x)` is the real part of `x` (See `%i` to enter complex numbers).
