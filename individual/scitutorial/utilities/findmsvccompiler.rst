====
"findmsvccompiler"
====

Scilab Function Last update : 28/02/2005
**findmsvccompiler** - detects Microsoft C Compiler



Calling Sequence
~~~~~~~~~~~~~~~~

msvc=findmsvccompiler()




Parameters
~~~~~~~~~~


+ **msvc** : returns 'msvc50','msvc60','msvc70','msvc71','unknown'




Description
~~~~~~~~~~~

detects Microsoft C Compiler.



Examples
~~~~~~~~


::

    msvc=findmsvccompiler()
