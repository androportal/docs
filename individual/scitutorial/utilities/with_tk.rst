Scilab Function Last update : 2002
**with_tk** - Checks if Scilab has been built with TCL/TK



Calling Sequence
~~~~~~~~~~~~~~~~

r=with_tk()




Parameters
~~~~~~~~~~


+ **r** : a boolean




Description
~~~~~~~~~~~

Returns **%t** if Scilab as been built with TCL/TK interface or **%f**
if not.
