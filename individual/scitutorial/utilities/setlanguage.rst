====
"setlanguage"
====

Scilab Function Last update : 25/10/2006
**setlanguage** - Sets the internal LANGUAGE value.



Calling Sequence
~~~~~~~~~~~~~~~~

setlanguage(language)




Parameters
~~~~~~~~~~


+ **language** : with language='fr' or 'eng'




Description
~~~~~~~~~~~

**setlanguage(language)** changes current language in scilab (help and
menus).
