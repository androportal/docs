====
"timer"
====

Scilab Function Last update : April 1993
**timer** - cpu time



Calling Sequence
~~~~~~~~~~~~~~~~

timer()




Description
~~~~~~~~~~~

Returns the CPU time from the preceding call to **timer()** .



Examples
~~~~~~~~


::

    
    
    timer();A=rand(100,100);timer()
