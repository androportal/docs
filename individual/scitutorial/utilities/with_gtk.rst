====
"with_gtk"
====

Scilab Function Last update : 2002
**with_gtk** - Checks if Scilab has been built with the "GIMP Toolkit"
library



Calling Sequence
~~~~~~~~~~~~~~~~

r=with_gtk()




Parameters
~~~~~~~~~~


+ **r** : a boolean




Description
~~~~~~~~~~~

Returns **%t** if Scilab as been built with gtk library interface or
**%f** if not.
