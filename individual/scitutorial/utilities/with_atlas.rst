====
"with_atlas"
====

Scilab Function Last update : 12/01/2005
**with_atlas** - Checks if Scilab has been built with Atlas Library



Calling Sequence
~~~~~~~~~~~~~~~~

r=with_atlas()




Parameters
~~~~~~~~~~


+ **r** : a boolean




Description
~~~~~~~~~~~

Returns **%t** if Scilab as been built with Atlas Library or **%f** if
not.
