


setstr (Matlab function)
========================

Set string flag



Matlab/Scilab equivalent
~~~~~~~~~~~~~~~~~~~~~~~~
**Matlab** **Scilab**

::

    setstr



::

    `ascii`_




Particular cases
~~~~~~~~~~~~~~~~

:S = setstr(A) with A a caracter string: In this case, Scilab
  **ascii** function convert string to ascii code matrix, but **setstr**
  keeps string format.
