====
"xbasr"
====

Scilab Function Last update : April 1993
**xbasr** - redraw a graphics window



Calling Sequence
~~~~~~~~~~~~~~~~

xbasr(win_num)




Description
~~~~~~~~~~~

**xbasr** is used to redraw the content of the graphics window with id
**win_num** . It works only with the driver **"Rec"** .
