"color"

Scilab Function Last update : 13/04/2004
**color** - returns the color id of a color



Calling Sequence
~~~~~~~~~~~~~~~~

id=color(name)
id=color(r,g,b)




Parameters
~~~~~~~~~~


+ **name** : name of a color.
+ **r,g,b** : RGB integer values of a color.
+ **id** : id of the color.

~~~~~~~~~~~
