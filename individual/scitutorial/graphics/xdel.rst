====
"xdel"
====

Scilab Function Last update : April 1993
**xdel** - delete a graphics window



Calling Sequence
~~~~~~~~~~~~~~~~

xdel([win-nums])




Parameters
~~~~~~~~~~


+ **win-nums** : integer or integer vector




Description
~~~~~~~~~~~

**xdel** deletes the graphics windows **win-nums** or the current
graphics window if no argument is given.
