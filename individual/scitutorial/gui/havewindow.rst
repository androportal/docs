====
"havewindow"
====

Scilab Function Last update : September 1994
**havewindow** - return scilab window mode



Calling Sequence
~~~~~~~~~~~~~~~~

havewindow()




Description
~~~~~~~~~~~

returns %t if scilab has it own window and %f if not, i.e. if scilab
has been invoked by "scilab -nw". ( **nw** stands for "no-window".
