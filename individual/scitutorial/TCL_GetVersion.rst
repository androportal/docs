


TCL_GetVersion
==============

get the version of the TCL/TK library at runtime.



Calling Sequence
~~~~~~~~~~~~~~~~


::

    TCL_GetVersion()
    ret=TCL_GetVersion('numbers')




Description
~~~~~~~~~~~

get the version of the TCL/TK library at runtime.

ret=TCL_GetVersion('numbers') returns a matrix with the version of the
TCL/TK library at runtime.



Examples
~~~~~~~~


::

    TCL_GetVersion() 
    TCL_GetVersion("numbers")
