


upper (Matlab function)
=======================

Convert string to upper case



Matlab/Scilab equivalent
~~~~~~~~~~~~~~~~~~~~~~~~
**Matlab** **Scilab**

::

    upper(str)



::

    `convstr`_(str,"u")




Particular cases
~~~~~~~~~~~~~~~~

If **A** is not a character string matrix, Scilab equivalent for
**B=upper(A)** is **B=A**, else equivalent is **B=convstr(A,"u")**.
