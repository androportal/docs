


lower (Matlab function)
=======================

Convert string to lower case



Matlab/Scilab equivalent
~~~~~~~~~~~~~~~~~~~~~~~~
**Matlab** **Scilab**

::

    lower(str)



::

    `convstr`_(str,"l")




Particular cases
~~~~~~~~~~~~~~~~

If **A** is not a character string matrix, Scilab equivalent for
**B=lower(A)** is **B=A**, else equivalent is **B=convstr(A)**.
