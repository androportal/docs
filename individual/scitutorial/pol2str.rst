


pol2str
=======

polynomial to string conversion



Calling Sequence
~~~~~~~~~~~~~~~~


::

    strs = pol2str(p)




Arguments
~~~~~~~~~

:p matrix of real polynomial
: :strs a matrix of strings
:



Description
~~~~~~~~~~~

converts polynomial to character string (utility function).
