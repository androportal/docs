====
"gspec"
====

Scilab Function Last update : April 1993
**gspec** - eigenvalues of matrix pencil (obsolete)



Calling Sequence
~~~~~~~~~~~~~~~~

[al,be]=gspec(A,E)
[al,be,Z]=gspec(A,E)




Description
~~~~~~~~~~~

This function is now included in the **spec** function. the calling
syntax must be replaced by


::

    
    
    [al,be]=spec(A,E)
    [al,be,Z]=spec(A,E)
