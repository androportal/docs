


null
====

delete an element in a list



Calling Sequence
~~~~~~~~~~~~~~~~


::

    l(i)=null()




Description
~~~~~~~~~~~

Deletion of objects inside a list.



Examples
~~~~~~~~


::

    l=`list`_(1,2,3);
    l(2)=null() // get list(1,3)
