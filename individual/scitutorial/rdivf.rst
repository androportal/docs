


rdivf
=====

right symbolic division



Calling Sequence
~~~~~~~~~~~~~~~~


::

    ["r"]=ldivf("d","c")




Arguments
~~~~~~~~~

:"d","c","r" strings
:



Description
~~~~~~~~~~~

returns the string `"c/d"` Trivial simplifications such as `"c/1" =
"c"` are performed.



Examples
~~~~~~~~


::

    `ldivf`_('c','d')
    `ldivf`_('1','2')
    `ldivf`_('a','0')
