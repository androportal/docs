


cd (Matlab function)
====================

Change/get working directory



Matlab/Scilab equivalent
~~~~~~~~~~~~~~~~~~~~~~~~
**Matlab** **Scilab**

::

    `cd`_



::

    `cd`_




Particular cases
~~~~~~~~~~~~~~~~

Note that **cd ..** does not work in Scilab, but it does in Matlab. In
Scilab you can use **cd("..")** or *cd ../*.
