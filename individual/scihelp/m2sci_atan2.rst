


atan2 (Matlab function)
=======================

Four-quadrant inverse tangent



Matlab/Scilab equivalent
~~~~~~~~~~~~~~~~~~~~~~~~
**Matlab** **Scilab**

::

    atan2



::

    atan2




Particular cases
~~~~~~~~~~~~~~~~

Matlab **atan2** function can work with complexes (in this case,
complex part is ignored), what Scilab **atan** can not.
