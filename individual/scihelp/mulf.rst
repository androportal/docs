


mulf
====

symbolic multiplication



Calling Sequence
~~~~~~~~~~~~~~~~


::

    mulf('d','c')




Description
~~~~~~~~~~~

returns the string `'c*d'` Trivial simplifications such as `'1*c' =
'c'` are performed.



Examples
~~~~~~~~


::

    mulf('1','a')
    mulf('0','a')
    'a'+'b'   //Caution...
