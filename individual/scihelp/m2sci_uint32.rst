


uint32 (Matlab function)
========================

Convert to 32-bit unsigned integer



Matlab/Scilab equivalent
~~~~~~~~~~~~~~~~~~~~~~~~
**Matlab** **Scilab**

::

    `uint32`_



::

    `uint32`_




Particular cases
~~~~~~~~~~~~~~~~

For infinite values, Scilab and Matlab **uint32** return different
values.



Examples
~~~~~~~~
**Matlab** **Scilab**

::

    A = `uint32`_(inf)
    A = 4294967295



::

    A = `uint32`_(%inf)
    A = 0
