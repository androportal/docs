


beep
====

Produce a beep sound



Calling Sequence
~~~~~~~~~~~~~~~~


::

    beep();
    beep('on')
    beep('off')
    s=beep()




Description
~~~~~~~~~~~

`beep()` produces your computer's default beep sound.

`beep('on')` turns the beep on

`beep('off')` turns the beep off

`s=beep()` returns the current beep mode (on or off).
