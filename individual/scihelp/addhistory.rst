


addhistory
==========

add lines to current history.



Calling Sequence
~~~~~~~~~~~~~~~~


::

    addhistory(string)
    addhistory(string_matrix)




Arguments
~~~~~~~~~

:string a string
: :string_matrix a string matrix
:



Description
~~~~~~~~~~~

add lines to current history.



Examples
~~~~~~~~


::

    addhistory('hello')
    addhistory(['hello','Scilab'])
