


Integer_pal
===========

Integer palette



Block Screenshot
~~~~~~~~~~~~~~~~





Module
~~~~~~


+ `xcos`_




Description
~~~~~~~~~~~

That palette is dedicated to handle integer numbers. Various basic
operators for the management of bit fields and for logic are
implemented as well as common gates encountered in digital circuits.
