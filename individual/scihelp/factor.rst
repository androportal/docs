


factor
======

factor function



Calling Sequence
~~~~~~~~~~~~~~~~


::

    [y]=factor(x)




Arguments
~~~~~~~~~

:x real scalar
: :y vector of primes numbers
:



Description
~~~~~~~~~~~

Given a real `x`, `factor(x)` returns in a vector `y` the primes
numbers decomposition of `x`. Particular cases: `factor(0)` returns
`0`, and `factor(1)` returns `1`.



Examples
~~~~~~~~


::

    x=620
    y=factor(x)
