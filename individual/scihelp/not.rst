


not
===

(~) logical not



Calling Sequence
~~~~~~~~~~~~~~~~


::

    ~A




Description
~~~~~~~~~~~

`~A` gives the element-wise negation of the elements of the boolean
matrix `A`.

~~~~~~~~


::

    ~[%t %t %f]
