


resethistory
============

Deletes all entries in the scilab history.



Calling Sequence
~~~~~~~~~~~~~~~~


::

    resethistory()




Description
~~~~~~~~~~~

Deletes all entries in the current scilab history.



See Also
~~~~~~~~


+ `savehistory`_ save the current history in a file
+ `loadhistory`_ load a history file
