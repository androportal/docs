


loadScicos
==========

Load the Xcos simulation native library.



Calling Sequence
~~~~~~~~~~~~~~~~


::

    loadScicos()




Module
~~~~~~


+ `xcos`_




Description
~~~~~~~~~~~

The Xcos simulation need a native shared library. This call force the
loading this library



Arguments
~~~~~~~~~

None



Example
~~~~~~~


::

    loadScicos();
