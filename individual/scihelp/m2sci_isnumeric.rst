


isnumeric (Matlab function)
===========================

Determine if input is a numeric array



Matlab/Scilab equivalent
~~~~~~~~~~~~~~~~~~~~~~~~
**Matlab** **Scilab**

::

    isnumeric(A)



::

    No equivalent




Particular cases
~~~~~~~~~~~~~~~~

There is no equivalent function for Matlab **isnumeric** function in
Scilab but it can be emuled by: **or(type(A)==[1 5 8])**.
