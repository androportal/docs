


dos (Matlab function)
=====================

Execute a UNIX command and return result



Matlab/Scilab equivalent
~~~~~~~~~~~~~~~~~~~~~~~~
**Matlab** **Scilab**

::

    dos



::

    `unix_g`_




Particular cases
~~~~~~~~~~~~~~~~

Output values order is inversed in Scilab and in Matlab.

In Scilab use **disp** to emulate Matlab **-echo** option.



Examples
~~~~~~~~
**Matlab** **Scilab**

::

    [status,result] = dos(...)



::

    [result,status] = `unix_g`_(...)
