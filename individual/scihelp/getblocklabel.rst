


getblocklabel
=============

Get the label of a scicos block



Module
~~~~~~


+ `xcos`_



::

    [label]=getblocklabel(blk)




Arguments
~~~~~~~~~


+ **blk :** Integer parameter. Set the index of a block (in the
  compiled structure).
+ **label :** String parameter. Gives the string of the label of the
  block numbered blk.
