


%i
==

imaginary unit



Calling Sequence
~~~~~~~~~~~~~~~~


::

    %i




Description
~~~~~~~~~~~

`%i` is imaginary unit, used to enter complex number.



Example
~~~~~~~


::

    1 + %i == `complex`_(1, 1)
          2 + 3 * %i == `complex`_(2, 3)




See Also
~~~~~~~~


+ `complex`_ Create a complex number.
+ `imult`_ multiplication by i the imaginary unitary
