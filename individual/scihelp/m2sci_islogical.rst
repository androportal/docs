


islogical (Matlab function)
===========================

Determine if item is a logical array



Matlab/Scilab equivalent
~~~~~~~~~~~~~~~~~~~~~~~~
**Matlab** **Scilab**

::

    islogical(A)



::

    No equivalent




Particular cases
~~~~~~~~~~~~~~~~

There is no equivalent function for Matlab **islogical** function in
Scilab but it can be emuled by: **or(type(A)==[4,6])**.
