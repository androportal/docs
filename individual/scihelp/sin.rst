


sin
===

sine function



Calling Sequence
~~~~~~~~~~~~~~~~


::

    [t]=sin(x)




Arguments
~~~~~~~~~

:x real or complex vector or matrix
:



Description
~~~~~~~~~~~

For a vector or a matrix, `sin(x)` is the sine of its elements. For
matrix sine use `sinm(X)` function.



Sample
~~~~~~



Examples
~~~~~~~~


::

    `asin`_(sin([1,0,%i]))




See Also
~~~~~~~~


+ `sinm`_ matrix sine function
+ `asin`_ sine inverse (radians)
