


more (Matlab function)
======================

Display Command Window output one screenful at a time



Matlab/Scilab equivalent
~~~~~~~~~~~~~~~~~~~~~~~~
**Matlab** **Scilab**

::

    more



::

    `lines`_




Particular cases
~~~~~~~~~~~~~~~~

See examples.



Examples
~~~~~~~~
**Matlab** **Scilab**

::

    more off
    more on
    more(30)



::

    `lines`_(0)
    `lines`_(60)
    `lines`_(30)
