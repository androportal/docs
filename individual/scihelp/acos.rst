


acos
====

element wise cosine inverse (radians)



Calling Sequence
~~~~~~~~~~~~~~~~


::

    t=acos(x)




Arguments
~~~~~~~~~

:x real or complex vector or matrix
: :t real or complex vector or matrix
:



Description
~~~~~~~~~~~

The components of `t` are cosine inverse of the corresponding entries
of `x`. Definition domain is [-1, 1].



Sample
~~~~~~



Examples
~~~~~~~~


::

    x=[1,%i,-1,-%i]
    `cos`_(acos(x))




See Also
~~~~~~~~
