


halt
====

stop execution



Calling Sequence
~~~~~~~~~~~~~~~~


::

    halt()
    halt('a message')




Description
~~~~~~~~~~~

stops execution until something is entered in the keyboard.



Examples
~~~~~~~~


::

    halt('Press a key')
    
    halt()




See Also
~~~~~~~~


+ `pause`_ pause mode, invoke keyboard
+ `return`_ return or resume execution and copy some local variables
+ `exec`_ script file execution
