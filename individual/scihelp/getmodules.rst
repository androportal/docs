


getmodules
==========

returns list of modules installed in Scilab



Calling Sequence
~~~~~~~~~~~~~~~~


::

    res=getmodules()




Arguments
~~~~~~~~~

:res a string matrix
:



Description
~~~~~~~~~~~

Returns list of modules installed in Scilab.
