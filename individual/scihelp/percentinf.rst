


%inf
====

infinity



Calling Sequence
~~~~~~~~~~~~~~~~


::

    %inf




Description
~~~~~~~~~~~

`%inf` returns the IEEE double representation for positive infinity.



Example
~~~~~~~


::

    A = %inf
          B = -%inf
          `isinf`_(A)
          `isinf`_(B)

~~~~~~~~
