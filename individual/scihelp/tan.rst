


tan
===

tangent



Calling Sequence
~~~~~~~~~~~~~~~~


::

    [t]=tan(x)




Arguments
~~~~~~~~~

:x vector or matrix
: :t vector or matrix
:



Description
~~~~~~~~~~~

The elements of `t` are the tangent of the elements of `x`.



Sample
~~~~~~



Examples
~~~~~~~~


::

    x=[1,%i,-1,-%i]
    tan(x)
    `sin`_(x)./`cos`_(x)




See Also
~~~~~~~~


+ `atan`_ 2-quadrant and 4-quadrant inverse tangent
+ `tanm`_ matrix tangent
