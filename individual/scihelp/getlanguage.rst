


getlanguage
===========

returns current language used by Scilab.



Calling Sequence
~~~~~~~~~~~~~~~~


::

    getlanguage()




Description
~~~~~~~~~~~

`getlanguage()` returns current language used by Scilab.



Examples
~~~~~~~~


::

    `setlanguage`_('en_US')
    getlanguage()




See Also
~~~~~~~~


+ `setlanguage`_ sets the internal LANGUAGE value (for Windows only)
