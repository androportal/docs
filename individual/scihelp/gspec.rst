


gspec
=====

eigenvalues of matrix pencil (obsolete)



Calling Sequence
~~~~~~~~~~~~~~~~


::

    [al,be]=gspec(A,E)
    [al,be,Z]=gspec(A,E)




Description
~~~~~~~~~~~

This function is now included in the `spec` function. the calling
syntax must be replaced by


::

    [al,be]=`spec`_(A,E)
    [al,be,Z]=`spec`_(A,E)

~~~~~~~~
