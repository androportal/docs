


fgets (Matlab function)
=======================

Read line from file, keep newline character



Matlab/Scilab equivalent
~~~~~~~~~~~~~~~~~~~~~~~~
**Matlab** **Scilab**

::

    fgets



::

    fgetstr




Particular cases
~~~~~~~~~~~~~~~~

Input parameter order is inversed in Scilab compared to Matlab.



Examples
~~~~~~~~
**Matlab** **Scilab**

::

    fgets(fid,n)



::

    fgetstr(n,fid)
