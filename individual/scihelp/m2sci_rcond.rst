


rcond (Matlab function)
=======================

Matrix reciprocal condition number estimate



Matlab/Scilab equivalent
~~~~~~~~~~~~~~~~~~~~~~~~
**Matlab** **Scilab**

::

    `rcond`_



::

    `rcond`_




Particular cases
~~~~~~~~~~~~~~~~

Scilab and Matlab values differ for empty matrix.



Examples
~~~~~~~~
**Matlab** **Scilab**

::

    c = `rcond`_([])
    c = Inf



::

    c = `rcond`_([])
    c = []
