**case** - keyword used in select

Description
~~~~~~~~~~~

Keyword used in ``         select ... case       ``

Use it in the following way:

::


    select expr0,
     case expr1 then instructions1,
     case expr2 then instructions2,
     ...
     case exprn then instructionsn,
     [else instructions],
    end
