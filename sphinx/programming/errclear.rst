**errclear** - error clearing

Calling Sequence
~~~~~~~~~~~~~~~~

``errclear([n])  ``

Description
~~~~~~~~~~~

clears the action (error-handler) connected to error of type
``         n       ``.

If ``         n       `` is positive, it is the number of the cleared
error ; otherwise all errors are cleared (default case)
