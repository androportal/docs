Scilab Function

Last update : April 1993

**abort** - interrupt evaluation.

Description
~~~~~~~~~~~

``         abort       `` interrupts current evaluation and gives the
prompt. Within a ``         pause       `` level
``         abort       `` return to level 0 prompt.
