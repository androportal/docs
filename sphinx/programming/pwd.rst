Scilab command

Last update : 23/06/2004

**pwd** - print Scilab current directory

**getcwd** - get Scilab current directory

Calling Sequence
~~~~~~~~~~~~~~~~

``pwd``

``x=pwd()  ``

``x=getcwd()  ``

Description
~~~~~~~~~~~

``         pwd       `` returns in ``         ans       `` the Scilab
current directory. ``         x=pwd()       `` or
``         x=getcwd()       `` returns in ``         x       `` the
Scilab current directory.

Examples
~~~~~~~~

::


    pwd
    x=pwd()
