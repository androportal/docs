**semicolon** - (;) ending expression and row separator

Description
~~~~~~~~~~~

In a file, the line separator \`\`;'' suppresses the display of the
line.

Within brackets ; denotes row separator in matrix definition.

Examples
~~~~~~~~

::

        a=[1,2,3;4,5,6];
        a=1;b=1,c=2
